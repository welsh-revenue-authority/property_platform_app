Features

The land and property data platform makes it simpler and faster for service delivery teams at the Welsh Revenue Authority and across Wales to use land and property data.

Is a property Wales or not?
Verify if a property is in Wales or not, or even if it crosses the border with England
Postcode and address lookup
Quickly implement a postcode and address lookup for your service
Tax zones
Check which Land Transaction Tax rules apply in different parts of Wales
Extended property information
Check if a property has registered for the tourism levy or contains a listed building
Link to other datasets
Reduced the effort of linking to other Welsh and UK government datasets

API access and downloads

The platform has a simple, well documented API you can integrate with your application or back-office system. Code libraries are available in popular languages to help you get started.

Some datasets, such as tax zones, are available as downloads in standard formats, such as GeoJSON and Frictionless Data.

Secure access for the public sector

Data that is sensitive or has licensing restrictions can only be accessed by organiations that should be accessing it. All data is marked as open, closed or shared so it's always clear how it can be used.
