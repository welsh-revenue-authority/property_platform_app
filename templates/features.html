Introduction

The land and property data platform makes it easier for service delivery teams in Wales to work with data about land and property.If you are designing and building public services you can use the platform to quickly access data.

It can also be used by analysts developing policies or researching outcomes.

API integration

the platform has a simple, well documented API you can integrate with your application or back-office system.
