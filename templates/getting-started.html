Getting started

The land and property data platform is for service delivery teams and data analysts at the Welsh Revenue Authority and wider public sector in Wales. Some information, such as tax zones, is available as open data and can be used by anyone.

Try the API now

You don't need to create an account to get started with our API. However, you will be asked to create an account to access some datasets where licensing restrictions apply. For some closed datasets, there will be additional verification steps for your account. Without an account, the API will return synthetic sample data, so you can still try it out.

Start exploring the API now

Pricing

The land and property platform is free to use unless you:

  • need access to certain datasets
  • make a very high number of requests
