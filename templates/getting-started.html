Getting started

The land and property data platform is for service delivery teams and data analysts working in the Welsh public sector. Some information is also available as open data and can be used by anyone.

You don't need to create an account to get started with our API.

Start exploring the API now

You will be asked to create an account to access some datasets where licensing restrictions apply. For some closed datasets, there will be additional verification steps for your account. Without an account, the API will return fake example data.

Pricing

The land and property platform is free to use unless you:

  • need access to certain datasets
  • make a very high number of requests
